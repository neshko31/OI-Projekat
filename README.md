# OI-Projekat

Projektni zadatak napravljen u sklopu predmeta OI na Fakultetu tehničkih nauka, Univerziteta u Novom Sadu.

U projektu je napisan programski kod koji kreira agenta koji igra "4 Connect" igru, upotrebom Minimax i Alpha-beta pruning algoritama.

Projekat je razvijan upotrebom python verzije 3.12.4
